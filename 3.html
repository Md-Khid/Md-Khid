Credit Card Billable Prediction Findings

This project employs an Artificial Neural Network to predict customer default risks within a dataset of credit facilities. It involves data processing in Python, addressing missing values and encoding categorical variables. Exploratory Data Analysis techniques are utilised to understand data distributions and identify outliers. The model’s performance is assessed using metrics such as accuracy, precision, recall, F1 score and AUC-ROC curve analysis. Additionally, the importance of predictors is evaluated to interpret the model’s decision-making process, thereby contributing to improved risk assessment strategies.
