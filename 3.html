Credit Card Billable Prediction Findings

This project integrates Python, SQL and R to streamline data preparation, focusing on data merging and cleansing. It utilises surveys to assess patient satisfaction through a Linear Model. Data is transferred from Excel to MySQL, forming precise database tables. SQL further refines these tables to ensure data consistency. R analyses this data, applying Linear Regression to predict satisfaction and identify influential variables. Finally, a performance impact chart is created to highlight areas for improvement, suggesting ward adjustments to enhance patient satisfaction.
