COVID-19 Country Grouping

The project utilises basic clustering methods such as K-means, leveraging IBM SPSS Modeller and JMP Statistical Software to categorise countries according to their COVID-19 risk levels. By conducting thorough data exploration and preparation, it examines important factors such as population demographics, GDP metrics and disease prevalence rates. Managing multicollinearity ensures the reliability of the clustering process thereby facilitating the identification of unique risk clusters. These clusters are invaluable for informing policy decisions, optimising resource allocation plans and implementing effective mitigation measures. This thorough analysis provides actionable insights essential for navigating the challenges posed by the ongoing pandemic.

 

Data Piping and Wrangling: Patient Survey Experience

The project demonstrates the seamless transfer of data among Python, SQL, and R for efficient data preparation, emphasizing data merging and cleansing. Utilising healthcare provider inpatient ward experience surveys, the project extracts insights using basic clustering and a simple Linear Model to predict patient satisfaction. Excel data is transferred to MySQL using Python, ensuring accurate table creation. SQL constructs a "doc_survey" table, handling missing values and transforming data types. R analyses the data, fitting a Linear Regression model to predict patient satisfaction with a focus on variable impact. A performance-impact chart highlights areas for improvement, recommending attention to ward type for enhanced patient satisfaction.
